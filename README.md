# PriceRadar Mobile

AI-Powered Price Intelligence & Analytics Mobile App built with Flutter.

## Features

- **Price Analyzer** - AI-powered product price analysis with market trends
- **Product Comparison** - Side-by-side AI comparison of products
- **Dynamic Pricing** - Intelligent pricing strategy generation
- **Smart Alerts** - AI-predicted price movement alerts
- **Material 3 Dark Theme** - Cyan/teal analytics-themed UI
- **8 Premium Screens** - Splash, onboarding, home, analyzer, comparison, dynamic pricing, alerts, settings

## AI Backend

- **Endpoint**: `sai.sharedllm.com/v1/chat/completions`
- **Model**: `gpt-oss:120b`
- **No API key required**

## Getting Started

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter 3.x with Material 3
- Dark theme with cyan/teal accent
- HTTP for AI API calls
- Provider for state management
